# Установка 

     apt update
     apt-get install screen
     git clone https://github.com/bednakovdenis/tghosting.git
     cd tghosting
     nano config.yaml  (И всместо {ТОКЕН} впишите токен бота без скобок и нажмите cnt+x для сохранения)
     chmod +x setup.sh
     screen -S bot
